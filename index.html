What is this?

This is pootis.party, a completley free and open source 24/7 web-radio. This radio is dedicated to playing TF2 music that YOU want to hear

Features

  • 24/7 Live stream
  • Free and open-source
  • Vote on the next song
  • Missing a song? Add it!
  • Vote on song submissions
  • Provides a stream URL that can be used with bots
